ALPHA BUILD PROJECT

TRIGGERS

Users hover mouse on the "I am not a robot" button

RULES

The microinteraction can only be seen when users hover over the button.

FEEDBACK

When users hover the button, the button's background change from black to white color and the text changes from white to black color.
When users move the mouse away, the button returns back to black color.

LOOPS

There are no loops in this microinteraction.
